# 🕯️ Aura Luz — Landing Page

## 📌 Sobre o Projeto

O **Aura Luz** é uma landing page desenvolvida com o objetivo de apresentar e divulgar produtos voltados ao bem-estar, como velas aromáticas, difusores e kits relaxantes.

O design foi pensado para transmitir sensações de **aconchego, tranquilidade e sofisticação**, utilizando uma paleta de cores quentes e elementos visuais inspirados em luz de velas e ambientes acolhedores.

---

## 🎯 Objetivo

Este projeto foi desenvolvido para:

* Praticar **HTML e CSS**
* Trabalhar conceitos de **design moderno (UI/UX)**
* Criar uma interface visualmente agradável e organizada
* Simular uma **landing page de e-commerce**

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3

---

## 🎨 Design

O layout segue um estilo:

* Minimalista
* Aconchegante
* Elegante

Com foco em:

* Boa hierarquia visual
* Tipografia limpa
* Cores suaves e harmônicas

---

## 📷 Funcionalidades
